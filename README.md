# q2a-verifyplus
<b>Plugin Name:</b>  Verify Plus <br>
<b>Price:</b> 30 USD (contact me to buy) <br>
<b>Plugin Description:</b> This is a premium-paid Q2A plugin that offers account verification (blue tick) for q2a users.<br>
<br>
<ul class="first">
	<b>Plugin Functionality:</b>
	<li>Plugin creates a table in database to store data.</li>
	<li>Admin can set minimum userlevel for moderator user (verifier) who can accept/reject verification request of other users (Default: Admin)</li>
	<li>Admin can allow/prevent users to make verification request</li>
	<li>Admin can enable/disable users to to upload verification file while making verification request (Max. File limit 2 MB)</li>
	<li>Admin can enable/disable point based-auto-verification. So, all users who reach certain points will be automatically verified by the system (Default: 1000 points)</li>
